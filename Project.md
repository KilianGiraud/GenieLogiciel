# Génie logiciel : Projet

Ce projet a été réalisé par Kilian Giraud, étudiant en M1 Informatique à l'ISIMA Aubière dans le cadre du cours de Génie logiciel.

Le but du projet étant de voir / maîtrise plusieurs notions :

* Differents types de tests :
    * Jacoco : Tests unitaires
    * Pitest : Tests de mutation de code
* Refactoring
* Ajout de fonctionnalités :
    * Correction de bugs
    * Ajout de classe
    * Ajout de gestion d'objet
    * Ajout d'une nouvelle méthode d'affichage en Markdown

## Tests / Refactoring / Features

Les parties effectués ainsi que des **Markdown explicatifs** sont disponibles dans les branches **tests | refactoring | features**.

Cette séparation a été faite dans le but de simplifier la gestion du projet sans avoir de conflit entre les codes.

## Conclusion

Ce projet aura été extrémement intéressant de par son importance en terme de fonctionnalités.

Il m'a permit de me rapprocher de mon but de maîtriser le développement logiciel.
